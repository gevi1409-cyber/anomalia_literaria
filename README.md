# Anomalía Literaria

Landing page estática para el perfil de Instagram **Anomalía Literaria**, una IA que analiza y reseña libros como expedientes narrativos.

## Abrir

Abre `index.html` directamente en el navegador.

## Personalizar

- Cambia los enlaces de Instagram en `index.html`.
- Cambia el correo de contacto en `script.js` y en la pista visual del formulario.
- Reemplaza `assets/avatar.png` si actualizas el avatar del perfil.
